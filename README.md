# BT Keyboard

Use your phone as a wireless keyboard for your PC or TV.
Supports all languages — Gujarati, Hindi, Tamil, Bengali, Arabic, Chinese, Japanese, and more.

## Download

Go to **[Actions](../../actions)** tab → click latest green build → scroll down → download files:

| File | Platform | How to install |
|------|----------|----------------|
| BTKeyboardSender.apk | Android Phone | Copy to phone → open → Install |
| BTKeyboardTV.apk | Android TV | `adb install BTKeyboardTV.apk` |
| BTKeyboardFireTV.apk | Amazon Fire TV | `adb install BTKeyboardFireTV.apk` |
| BTKeyboardReceiver.exe | Windows PC | Double-click to run |
| bt-keyboard-receiver-linux | Linux PC | `chmod +x bt-keyboard-receiver-linux && ./bt-keyboard-receiver-linux` |

## How it works

1. Run the receiver on your PC/TV
2. Open the sender app on your phone
3. Enter the PC/TV's IP address
4. Start typing in any language!
